# Smart Health Risk Predictor

An intelligent health risk prediction system built with Python, Flask, Machine Learning, and MongoDB. The system analyzes user health parameters and predicts potential health risks using trained ML models.

---

## 🚀 Features
* **User Authentication:** Secure signup and login for users.
* **ML Predictions:** Real-time risk assessment using optimized Machine Learning models.
* **Database Management:** Efficient storage of health records and user profiles using MongoDB.
* **Responsive Frontend:** Clean and interactive user interface.

---

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** Python (Flask, Flask-CORS, JWT)
* **Database:** MongoDB
* **Machine Learning:** Scikit-learn, Pandas, NumPy

---

## ⚙️ Installation & Setup

### 1. Prerequisites
Make sure you have MongoDB installed locally and running on port `27017`.

### 2. Backend Setup
Navigate to the backend directory, activate the virtual environment, install dependencies, and run the server:

```bash
cd backend
venv\Scripts\activate
pip install -r requirements.txt
python app.py
